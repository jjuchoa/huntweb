instalações necessárias:
(Node.js, NPM, Yarn)

Configuração inicial

1 - instalar o create-react-app de forma global (-g)

npm install -g create-react-app

2 - criar um nome para o projeto

create-react-app <nome do projeto>
ex.:
create-react-app huntweb

3 - abrir a pasta do projeto no VS Code

4 - rodar o projeto

npm start
